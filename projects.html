Uncertainty-Aware Meta-Learning for Robust Stent Segmentation via Selective Ensemble

Asan Institute of Life Sciences / Asan Medical Center 2025 – Present
Deep Learning Segmentation Selective Ensemble Meta-Learning CinC 2026

Lead Researcher — Stent Registration & Segmentation Pipeline

Deep learning pipeline for stent and catheter segmentation in invasive coronary angiography, supporting post-PCI QCA and stent-vessel registration. Separate models handle fluoroscopy (catheter body & tip) and angiography (catheter body/tip + stent) using U-Net++, FPN, and Segformer with EfficientNet-B7 encoders. Stent-vessel registration uses the catheter tip as a spatial reference via ICP, with auxiliary midpoint-to-centerline alignment for robustness against tip prediction errors.

Problem

Single models with different loss functions show distinct failure modes — disconnected vessels, spurious branches, boundary errors — making naive ensembling unreliable for post-PCI QCA.

Approach

  • Five U-Net / DenseNet-121 baselines with morphology-focused losses
  • MSEN & ESEN rank-based selective ensemble
  • Attention meta-learner with uncertainty maps

Results

Up to 5.04% DSC improvement over weakest baseline (avg. 3.68%). Evaluated on 942 internal + 358 external PCI projections with five-fold cross-validation.

Publication: Uncertainty-Aware Meta-Learning for Robust Stent Segmentation via Selective Ensemble — Computing in Cardiology (CinC) 2026

MDCKE: Multimodal Deep-Context Knowledge Extractor that Integrates Contextual Information

School of Computing, Gachon University 2023 – 2025
Multimodal NER Relation Extraction Knowledge Graph Image Captioning BERT NLP

First Author — Full pipeline design, implementation, and evaluation

MDCKE is an information extraction framework for constructing multimodal knowledge graphs that reflect deeper contextual understanding. It generates hierarchical multi-scale image representations (global + local) and captions via an Image Transformer, then fuses them with text using a dynamic-gate visual prefix that injects image features into every BERT self-attention layer. A CRF-based classifier jointly performs Named Entity Recognition (NER) and Relation Extraction (RE) to output structured entity-relation triples.

Key Idea

Captions bridge the modality gap: global and local image captions are constituency-parsed and spliced with text, supplying semantic context that pure visual features miss — especially under low-resource conditions.

Architecture

  • ResNet-101 + FPN pyramid for multi-scale visual features
  • Image Transformer captioning (global & local crops)
  • Dynamic-gate visual prefix fused into each BERT layer
  • CRF head for joint NER & RE

Results

Twitter2015 NER: F1 82.44% (+5.83% over SOTA). MNRE RE: F1 88.30% (+26.26% over SOTA). Maintains consistent gains in 10–50% low-resource settings where prior models degrade sharply.

Publication: MDCKE: Multimodal Deep-Context Knowledge Extractor that Integrates Contextual Information — Alexandria Engineering Journal, Elsevier, 2025

Automated Medical Report Generation for Coronary Angiography

MLLM LoRA Fine-tuning RAG LangGraph Multi-frame CDSS

End-to-end CDSS pipeline generating structured clinical reports from coronary angiography. A two-stage LoRA fine-tuning pipeline first personalizes the VLM's reporting style, then RAG retrieves semantically similar cases from a vector DB built on DCM images, FFR tables, and historical reports. A LangGraph-based parallelized multi-agent system handles vessel classification, finding extraction, and report assembly. Multi-frame input with a TSAU-based temporal attention classifier enables multi-view vessel analysis and vessel-type–conditioned report generation.

Coronary Centerline Extraction

Centerline Morphology QCA

Geometric centerline extraction from vascular segmentation masks. A skeletonized centerline's average slope segments the mask into entrance/exit regions; perpendicular normals are drawn at 3 px intervals, and each normal's contour intersections are used to shift the centerline point to the true vessel midline. The refined centerline feeds directly into QCA diameter measurement and stent-vessel registration.
